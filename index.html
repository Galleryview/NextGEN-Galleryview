NextGEN-Galleryview

NextGEN-Galleryview (originally created by Alex Rabe) is a Wordpress plugin providing NextGen-Gallery plugin users a new template based on the Galleryview jQuery Plugin created by Jack Wanders (@jackwanders). The plugin has been updated by Brandon Hubbard (@bhubbard).

Basic Usage:

Simply use galleryview as your NextGen-Gallery Template.

[nggallery id=1 template="galleryview"]

Advanced Usage:

For advanced use its recommended to create your own nextgen-gallery template.

  1. Create a folder called nggallery within your Wordpress theme.
  2. Copy gallery-galleryview.php from the "view" folder of the plugin and place it within the nggallery folder you created.
  3. Now rename the file to gallery-ANAMEYOULIKE.php
  4. Now modify the template to your desires and use with NextGenGallery. 
    [nggallery id=1 template="ANAMEYOULIKE"]
